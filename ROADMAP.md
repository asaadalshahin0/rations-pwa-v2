# Rations Roadmap

## MVP

- Log meals from photo or text.
- Track calories and macros against daily goals.
- Review local meal history.
- Sync encrypted data across devices using a private phrase.
- Persist user food memory for recurring foods, brands, portions, and macros.

## Done

- Add a compact streak counter for protein-plus-calorie goal days.
- Validate the new streak UX in the running PWA.
- Add persistent food memory to AI estimates and encrypted sync data.

## Later

- Keep future changes surgical and tied to explicit user requests.
